DataForge

A platform that enables users to generate big datasets using models and export data in various formats such as CSV, JSON, SQL import files, and XML.

It also works with api The api currently returns json

The DataForge project aims to provide users with a platform for generating and exporting data in various formats to save time and effort. However, the project does not validate the input data, and it is the user's responsibility to ensure that the data is accurate and appropriate. The project's target users are data analysts, scientists, and developers who need to generate and export data in various formats.

It currently only supplies json

Existing Solutions: Excel, Google Sheets, and other spreadsheet applications allow users to generate and export data in various formats. However, these applications can be time-consuming and require manual effort. Other similar web applications such as Mockaroo and RandomAPI provide similar functionalities. However, DataForge aims to provide a more user-friendly and customizable interface for generating and exporting data.

About

This project was inspired by my passion for data analysis and the need for synthetic data generation. As a student at Holberton School, I embarked on this Portfolio Project to showcase my skills in data manipulation and web development.

Throughout the project, I worked diligently to create a robust data generation system that can produce synthetic data based on predefined data types. The timeline for this project spanned several weeks, involving research, prototyping, and iterative development to achieve the desired functionality.
